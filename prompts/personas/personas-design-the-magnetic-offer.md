# Personas - Design the Magnetic Offer


---

Role:
Act like a senior marketing strategist and offer architect.

Objective:
Design a “Magnetic Offer” for one primary customer persona using the model: Pain → Dream → Blockers → Upgrade → Magnetic Angle.

Inputs you will receive (between triple dashes):
--- 
Persona snapshot, internal performance/CRM data, and voice-of-customer artifacts (surveys, reviews, NPS notes). 
---

Process (step-by-step):
1) Map Current State: Extract the top 3 pains felt most acutely by this persona (evidence-tag each with a source).
2) Define the Dream: Describe the “after” state in specific, outcome-centered terms (quantify when possible).
3) Identify Blockers: List practical, emotional, and environmental obstacles preventing progress (rank by severity).
4) Frame the Upgrade: Propose offer components that remove blockers (deliverables, risk-reversal, proof, bonuses, success metrics).
5) Craft the Magnetic Angle: Articulate the big idea + primary promise + unique mechanism + sharp differentiation.
6) Validate: Cross-check each element against inputs; rewrite anything not grounded in evidence or clear logic.
7) Self-Check: Confirm one persona only, each element ≤50 words, no jargon, measurable where possible.

Output Format (choose one):
A) Table with columns: Stage | Summary (≤50 words) | Evidence Tag(s)
B) Structured paragraph with clear Stage labels; each label followed by ≤50 words; include Evidence Tag(s).

Constraints & Stop Conditions:
- Use exactly one primary persona per run.
- Keep every stage under 50 words.
- If a required input is missing, state explicit assumptions in an “Assumptions (≤40 words)” line and proceed.
- Do not add extra sections beyond the chosen output format and optional Assumptions line.

Quality Bar:
- Be precise, concrete, and testable.
- Use the persona’s language (mirror phrases from surveys/reviews/NPS when cited).
- Prefer numbers, timeframes, and acceptance criteria over adjectives.
- End with a one-sentence “Next Test” suggesting the smallest, fastest experiment to validate the angle.

Take a deep breath and work on this problem step-by-step.
