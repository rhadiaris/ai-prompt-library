# Landing Page Creator


---

Act as a Direct Response Copywriter.
I am building a landing page for my High-Ticket Offer.

My Offer: [Insert Offer Name, e.g., The 2-Hour Creator System]

My Ideal Client: [Insert Avatar, e.g., Burnt out corporate moms]

The Problem: [Insert Pain, e.g., They have no time and want extra income]

The Solution: [Insert Solution, e.g., A system to build a newsletter in nap times]

Task: Write the copy for a high-converting landing page using the following structure:

1. The Headline: A bold, benefit-driven hook that promises a specific result (use numbers).

2. The Subheadline: Call out the “Enemy” or the “Old Way” they are currently doing.

3. The “Paradigm Shift” Table: Create a comparison table text: “Life Before [My Offer]” vs “Life After [My Offer].” Focus on emotional and time-based differences.

4. The “Who This Is For” (The Filter): A bulleted list of 3 types of people who should buy, and 3 types who should not buy (repel the freebie seekers).

5. The “Stack”: A bulleted list of the deliverables, but rewritten as “Benefits,” not “Features.” (e.g., instead of “5 Videos,” write “The System to save 10 hours a week”).

Generate 5 Hero Section variations — one for each psychological angle listed above

Bold the most compelling phrase in each framework description

Use specific numbers wherever possible

Prioritize clarity and a compelling promise in headlines — length doesn't matter as much as the strength of the hook

Tone: Authoritative, empathetic, direct. No fluff.
