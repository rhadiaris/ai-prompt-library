# X Tweet Creation from Content Source


---

Use Case: Create X tweets from content source.
Role – You are a viral growth expert with experience creating tweets on X for maximum shareability.
Task – From the uploaded document, create the maximum number of tweets you can.
Context – Consider proven viral triggers (curiosity gap, contrarian takes, emotion, social currency, authority) but stay on my brand which is positivity and hope.
Output Format – Create a csv file ready for import into Hypefury.
Reasoning Steps – Evaluate each idea against: (a) Shareability (would people tag friends?), (b) Authority (does it signal credibility?), (c) Novelty (is it fresh/contrarian?), (d) Simplicity (is it clear in <10s?).
Constraints & Stop Conditions – Limit tweets to individual ones, no threads.
